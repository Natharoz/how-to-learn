## Description:

"How to learn" is a one-page website about learning methodologies. The site is created using BEM Nested file structure.

## Technology Stack:

- HTML5;
- CSS3:
  - Flexbox;
  - Positioning elements;
  - Animations;
- BEM methodology;
  - file Structure and File Paths (Nested BEM);
  - block reuse.
